# 집에서 실행하는 방법

## 가장 중요한 점

압축 파일 안의 내용물을 **폴더째 그대로 압축 해제**해야 합니다.
`app.py` 같은 파일 하나만 옮기면 실행되지 않습니다.

앱은 코드뿐 아니라 아래 파일들을 함께 사용합니다.

- `app/`, `forest_reco/`, `scripts/`
- `requirements.txt`
- `data/` 안의 경량 실데이터
- `앱_실행.bat`, `start_app.ps1`

## 실행 순서

1. 압축 파일을 원하는 위치에 풉니다.
2. 폴더 안의 `앱_실행.bat`을 더블클릭합니다.
3. 검은 창이 열리면 닫지 말고 기다립니다.
4. 브라우저가 자동으로 열리면 앱을 사용합니다.

처음 실행할 때 필요한 파이썬 패키지가 없으면 자동 설치를 시도합니다.
이 과정은 몇 분 걸릴 수 있습니다.

## 미리 설치해야 하는 것

집 컴퓨터에 Python이 없다면 먼저 Python 3.12를 설치하세요.
설치할 때 **Add python.exe to PATH** 체크가 필요합니다.

다운로드: https://www.python.org/downloads/

## 휴대폰에서 보기

PC와 휴대폰이 같은 와이파이에 있으면, 실행 창에 표시되는
`http://PC-IP:8501` 주소를 휴대폰 브라우저에 입력하면 됩니다.

Windows 방화벽 창이 뜨면 허용해야 휴대폰 접속이 됩니다.

## 그래도 안 될 때 확인할 것

- 압축 안의 파일을 일부만 꺼낸 것이 아닌지 확인
- `data/` 폴더가 같이 있는지 확인
- Python 3.12가 설치되어 있는지 확인
- 처음 실행 시 인터넷이 연결되어 있는지 확인

이번 전달본은 코드와 실데이터를 함께 묶는 용도입니다.
원본 raw 데이터는 너무 커서 제외했고, 앱 실행에는 경량화된 데이터만 있으면 됩니다.
